# Billing Made Easy – README [page:1]

Billing Made Easy is a simple web-based billing helper that lets you add items, apply tax, and see totals update instantly.[page:1]

## Overview

This page provides a lightweight billing interface where you can enter line items with cost, quantity, and tax percentage, and automatically get item totals, tax totals, and the grand total.[page:1] It is designed to be quick to use for small businesses, freelancers, or personal invoices.[page:1]

## Features

- Live summary of Item Total, Tax Total, Grand Total, and Items Count at the top of the page.[page:1]
- Item entry fields: Item name, Cost, Qty, and Tax % for each line.[page:1]
- Buttons to **ADD** ITEM and **DELETE LAST** to manage the list quickly.[page:1]
- Editable table where you can click a cell to change values, with Delete key support for removing the last item.[page:1]

## How to Use

1. Enter the **Item name**, **Cost**, **Qty**, and **Tax %** in the input fields.[page:1]
2. Click the **ADD ITEM** button to add the line to the billing table.[page:1]
3. Repeat for all items you need to bill; the summary section will update totals automatically.[page:1]
4. To remove the most recent item, click **DELETE LAST** or press the **Delete** key as indicated.[page:1]
5. Use the editable table to adjust any cell if you need to correct an entry.[page:1]

## Layout

- Header: Title “Billing Made Easy”.[page:1]
- Summary panel: Item Total, Tax Total, Grand Total, and Items Count, each showing a dollar value or count.[page:1]
- Controls: Input fields for item details plus **ADD ITEM** and **DELETE LAST** buttons.[page:1]
- Items table: Columns for `#`, `Item Name`, `Cost`, `Qty`, `Tax %`, `Line Total`, and `Actions`.[page:1]
- Footer: “Billing Made Easy © – Click a cell to Edit. Delete key removes last.”[page:1]

## Requirements

This is a standalone HTML page that runs in any modern web browser; no installation is needed.[page:1]
