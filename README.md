# Billing Made Easy – README  

Billing Made Easy is a simple web-based billing helper that lets you add items, apply tax, and see totals update instantly. 

## Overview

This page provides a lightweight billing interface where you can enter line items with cost, quantity, and tax percentage, and automatically get item totals, tax totals, and the grand total.  It is designed to be quick to use for small businesses, freelancers, or personal invoices. 

## Features

- Live summary of Item Total, Tax Total, Grand Total, and Items Count at the top of the page. 
- Item entry fields: Item name, Cost, Qty, and Tax % for each line. 
- Buttons to **ADD** ITEM and **DELETE LAST** to manage the list quickly. 
- Editable table where you can click a cell to change values, with Delete key support for removing the last item. 

## How to Use

1. Enter the **Item name**, **Cost**, **Qty**, and **Tax %** in the input fields. 
2. Click the **ADD ITEM** button to add the line to the billing table. 
3. Repeat for all items you need to bill; the summary section will update totals automatically. 
4. To remove the most recent item, click **DELETE LAST** or press the **Delete** key as indicated. 
5. Use the editable table to adjust any cell if you need to correct an entry. 

## Layout

- Header: Title “Billing Made Easy”. 
- Summary panel: Item Total, Tax Total, Grand Total, and Items Count, each showing a dollar value or count. 
- Controls: Input fields for item details plus **ADD ITEM** and **DELETE LAST** buttons. 
- Items table: Columns for `#`, `Item Name`, `Cost`, `Qty`, `Tax %`, `Line Total`, and `Actions`. 
- Footer: “Billing Made Easy © – Click a cell to Edit. Delete key removes last.” 

## Requirements

This is a standalone HTML page that runs in any modern web browser; no installation is needed.
